Multi-agent AI framework for automating key stages of the Software Development Lifecycle (SDLC).  

This project presents a workflow-orchestrated agentic system that transforms natural language requirements into structured software artefacts through coordinated AI agents. The framework integrates requirements analysis, code generation, validation, orchestration, and deployment analysis within a single end-to-end pipeline.

The system demonstrates process-level automation of the SDLC, enabling traceability from initial requirements through to generated code, validation outputs, and deployment configurations. It highlights the potential of agentic AI to enhance productivity, consistency, and early-stage software engineering workflows.
