## Agentic SDLC Framework

A multi-agent AI system for automating the Software Development Lifecycle (SDLC).

This project presents a workflow-orchestrated agentic framework that transforms natural language requirements into structured software artefacts through coordinated AI agents. The system integrates requirements analysis (A1), code generation (A2), validation (A3), orchestration (A4), and deployment analysis (A6) into a unified pipeline.

The framework demonstrates process-level SDLC automation, enabling end-to-end traceability from requirements to implementation and deployment readiness. It highlights the potential of agentic AI to support scalable, consistent, and structured software engineering workflows.
